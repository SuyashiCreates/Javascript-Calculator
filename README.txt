# JAVASCRIPT CALCI


##OVERVIEW


 This is a simple web based calculator built using Javascript  to perform different arithmetic operation and help user quickly and efficiently.


##Usage
 We can simply perform various calculation by enetring various expression and using different arithmetic operators.

##Special Feature
  Our JAVASCRIPT CALCI PROVIDES A FEATURE TO LOAD THE HISTORY OF THE USER'S  PREVIOUS CALCULATION THIS MIGHT HELP USER TO PERFORM COMPLEX OPERATION. IT ALSO PROVIDE AN EFFECT OF TRANSITION BEFORE GETTING DISPLAYED ON SCREEN.
